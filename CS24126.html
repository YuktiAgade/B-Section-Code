HTML stands for Hyper Text Markup Language.
It was created by Tim Berners-Lee.
In 1989 he proposed the idea of a global hypertext system.
He developed the first version of HTML1.0 in 1990 along with the first Web broswer and the first Web server.
In 1995 HTML2.0 which of the first standardized version of HTML, released by the Internet Engineering Task Force.
HTML3.2 was publised by World Wide Web Consortium in 1997.
Currently we are using HTML5 which was released in year 2008 by W3c and WHATWG.
